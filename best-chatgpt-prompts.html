Writing Prompts

Transform your writing with these battle-tested prompts.

Blog Post Generator Writing

You are an expert content writer specializing in [NICHE]. Write a comprehensive blog post about "[TOPIC]". Requirements: - Engaging hook in the introduction - 5-7 main sections with subheadings - Include practical examples and actionable tips - Conversational yet authoritative tone - End with a compelling call-to-action Target audience: [DESCRIBE READER] Word count: Approximately [NUMBER] words

Email Writer Writing

Write a professional email for the following situation: Purpose: [STATE YOUR GOAL] Recipient: [WHO ARE THEY] Tone: [professional/friendly/urgent/apologetic] Key points to include: - [POINT 1] - [POINT 2] - [POINT 3] Include a clear subject line and call-to-action.

Content Improver Writing

Improve the following text while maintaining the original meaning. Focus on: 1. Clarity and readability 2. Stronger word choices 3. Better sentence flow 4. Eliminating redundancy Original text: """ [PASTE YOUR TEXT] """ Provide the improved version and briefly explain the key changes you made.

Coding Prompts

Level up your development workflow with these coding prompts.

Code Explainer Coding

Explain the following code in detail: ```[LANGUAGE] [PASTE CODE] ``` Please provide: 1. A high-level summary of what this code does 2. Line-by-line breakdown of the logic 3. Any potential issues or improvements 4. Example use cases Explain as if teaching a developer who knows the basics but is unfamiliar with this specific implementation.

Debug Assistant Coding

Help me debug this code: Code: ```[LANGUAGE] [PASTE CODE] ``` Error message: [PASTE ERROR] Expected behavior: [WHAT SHOULD HAPPEN] Actual behavior: [WHAT HAPPENS INSTEAD] Please: 1. Identify the root cause 2. Explain why this error occurs 3. Provide the corrected code 4. Suggest how to prevent similar issues

Code Generator Coding

Write [LANGUAGE] code that: Functionality: [DESCRIBE WHAT IT SHOULD DO] Input: [DESCRIBE INPUTS] Output: [DESCRIBE EXPECTED OUTPUT] Constraints: [ANY LIMITATIONS OR REQUIREMENTS] Requirements: - Include error handling - Add clear comments - Follow best practices for [LANGUAGE] - Make it production-ready

Business Prompts

Drive business results with these professional prompts.

SWOT Analysis Business

Conduct a comprehensive SWOT analysis for my business: Business: [NAME/DESCRIPTION] Industry: [INDUSTRY] Current situation: [BRIEF OVERVIEW] Analyze: - Strengths (internal advantages) - Weaknesses (internal challenges) - Opportunities (external possibilities) - Threats (external risks) For each point, provide specific actionable recommendations.

Business Plan Section Business

Write the [SECTION NAME] section of a business plan for: Business: [DESCRIPTION] Target market: [WHO] Unique value: [WHAT MAKES IT DIFFERENT] Make it: - Professional and investor-ready - Data-driven where possible - Realistic but compelling - Approximately [LENGTH] words

Marketing Prompts

Create compelling marketing content that converts.

Ad Copy Generator Marketing

Create compelling ad copy for: Product/Service: [WHAT YOU'RE SELLING] Target audience: [WHO] Platform: [Facebook/Google/Instagram/LinkedIn] Goal: [awareness/clicks/conversions] Provide: - 3 headline variations (under 30 characters each) - 3 description variations - Primary text/body copy - Call-to-action options Focus on benefits, not features. Use emotional triggers.

Social Media Content Marketing

Create a week's worth of social media content for: Brand: [NAME] Industry: [INDUSTRY] Voice: [DESCRIBE BRAND VOICE] Platform: [PLATFORM] For each day, provide: - Post copy (platform-appropriate length) - Hashtag suggestions - Best time to post - Engagement hook or question Mix of content types: educational, entertaining, promotional, user-generated.

Productivity Prompts

Get more done with AI-powered productivity.

Meeting Summary Productivity

Summarize these meeting notes into a professional summary: """ [PASTE MEETING NOTES] """ Include: 1. Meeting objective 2. Key discussion points (bullet points) 3. Decisions made 4. Action items with owners and deadlines 5. Next steps Format for easy scanning and sharing with stakeholders.

Task Breakdown Productivity

Break down this project into actionable tasks: Project: [DESCRIBE PROJECT] Deadline: [DATE] Team size: [NUMBER] Constraints: [ANY LIMITATIONS] Provide: - Phase breakdown - Individual tasks with time estimates - Dependencies between tasks - Milestones and checkpoints - Risk factors and mitigation strategies

Creative Prompts

Unlock your creativity with these imaginative prompts.

Story Starter Creative

Write the opening scene of a [GENRE] story with these elements: Setting: [WHERE/WHEN] Main character: [BRIEF DESCRIPTION] Mood: [ATMOSPHERE] Hook: [WHAT DRAWS READERS IN] Requirements: - Start in the middle of action - Establish voice immediately - Include sensory details - End the scene with tension or intrigue Approximately 500 words.

Brainstorm Ideas Creative

Generate 10 unique ideas for [TOPIC/PROJECT]. Context: [BACKGROUND INFO] Constraints: [ANY LIMITATIONS] Style: [INNOVATIVE/PRACTICAL/BOLD/SAFE] For each idea, provide: - The concept in one sentence - Why it could work - Potential challenges - First step to implement Think outside the box. Include at least 2 unconventional ideas.
